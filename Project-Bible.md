## 二、核心概念

### 2.1 一句话卖点
> **无灵根杂役误入诡仙界，唯凡人之躯可破禁忌，一路逆伐成仙。**

### 2.2 主角设定

| 项目 | 内容 |
|-----|------|
| **姓名** | 林守一 |
| **年龄** | 19岁（故事开始时） |
| **身份** | 青云宗外门杂役弟子，无灵根、无背景、无资源的"三无"修士 |
| **核心优点** | 极致谨慎、冷静自持、坚韧不屈 |
| **致命缺陷** | 过度不信任、执念深重、自我封闭 |
| **核心恐惧** | 无意义的死亡、被遗忘 |
| **终极欲望** | 证明凡人的价值 |
| **成长弧光** | 封闭 → 尝试信任 → 为他人而战 |
| **台词特征** | 言简意赅、不带情绪、善用反问、偶尔自嘲 |

### 2.3 金手指/外挂

| 项目 | 内容 |
|-----|------|
| **名称** | 规则免疫 |
| **获得方式** | 天生（无灵根体质）+ 意外（误入诡仙界） |
| **核心能力** | Lv1规则无视 → Lv2规则感知 → Lv3规则传递 → Lv4规则破解 |
| **使用代价** | 凡人脆弱、精神消耗、传递削弱、暴露风险 |
| **成长路径** | 窥规者 → 测规者 → 渡规者 → 解规者 → 逆规者 |
| **上限天花板** | 逆规者境界；无法正面对抗诡神；无法破坏创世规则 |

### 2.4 核心冲突

**外部冲突**：
- Lv1：主角 vs 具体规则（生存）
- Lv2：主角 vs 诡异生物（战斗）
- Lv3：主角 vs 诡神（解放）

**内部冲突**：凡人的局限性 vs 仙道的诱惑（是否诡化以获得力量）

**终极对手**：虚无诡神（归墟之主）——诡仙界最高层级的诡神，所有规则的源头

## 六、关键设定台账

### 6.1 市场分析结论 [S01-BATCH-01]

**平台选择参考**：
- 起点中文网：付费订阅，用户付费意愿高，适合精品长篇
- 番茄小说：免费+广告，流量最大，适合爽文快餐
- 七猫免费小说：免费+广告，下沉市场为主

**热门品类分布**（TOP150统计）：
- 都市：30.7%
- 玄幻：22.7%
- 言情：13.3%
- 仙侠：9.3%
- 悬疑：5.3%

**上升最快的3个新兴类型**：
1. 灵气复苏/末世异能
2. 无限流/副本流
3. 诡异/规则怪谈

**标杆作品分析**：
1. 《诡秘之主》- 世界观深度标杆
2. 《夜的命名术》- 幽默文风标杆
3. 《斗破苍穹》- 废柴逆袭模板鼻祖
4. 《大奉打更人》- 架空历史+权谋装逼标杆
5. 《我有一剑》- 暴力美学+极致剑道标杆

### 6.2 市场机会缺口 [S01-BATCH-02]

**过度饱和元素（避开）**：
1. 传统系统流/签到流
2. 纯正赘婿/龙王/战神归来
3. 无限流的传统副本模式

**供给不足方向（机会）**：
1. 冷门历史时期（五代十国、南北朝等）
2. 职业深度垂直（冷门职业+专业知识）
3. 逆向都市异能（能力消退/代价设定）

**创新组合机会**：
1. 规则怪谈 × 职业背景
2. 灵气复苏 × 传统文化复兴（首选方向）
3. 无限流 × 中国特色背景

### 6.3 创意方向A - 品类选择 [S01-BATCH-02]

**确定品类**：都市异能 + 灵气复苏 + 传统文化元素

**细分定位**：以"非遗传承者觉醒异能"为核心设定

**选择理由**：
- 市场容量大：都市类占比30.7%，灵气复苏增长150%
- 差异化明显：传统文化觉醒几乎是空白领域
- 平台友好：文化自信叙事符合主流价值观

### 6.4 创意方向A - 核心创意设计 [S01-BATCH-03]

**一句话卖点**：落魄皮影师傅灵气觉醒，操控万千光影，守护非遗传承逆袭都市。

**主角设定**：
- 姓名：陆寻影
- 身份：陕西华县皮影世家第七代传人
- 核心欲望：让皮影艺术重获新生
- 致命弱点：固执、自卑、偏执（对"正宗传承"有洁癖）

**金手指设计**：「影戏术」
- 获得方式：灵气复苏时与十五年雕刻功底融合
- 核心能力：操控亲手雕刻的皮影实体化作战
- 使用代价：精神消耗、皮影损耗、光源依赖
- 成长路径：灯影师 → 戏影师 → 映世师 → 影圣

**核心冲突**：
- 外部冲突：主角 vs 「光辉教」（反传统文化邪教组织）
- 内部冲突：传承的纯正性 vs 时代的必要妥协

### 6.5 创意方向B - 品类选择 [S01-BATCH-03]

**确定品类**：历史架空 + 五代十国背景 + 商战权谋

**细分定位**：以五代十国乱世商人为主角的历史架空小说

**选择理由**：
- 市场规模中等但用户质量高，付费意愿强
- 五代十国题材占比<1%，差异化空间极大
- 商人视角打破"皇帝/将军/谋士"三板斧

### 6.6 创意方向B - 核心创意设计 [S01-BATCH-03]

**一句话卖点**：破产富商穿越五代乱世，以商道布局天下，从贩夫走卒到商界帝王。

**主角设定**：
- 姓名：秦商（原名秦朗）
- 身份：现代破产商人穿越为乱世流民
- 核心欲望：建立无法被撼动的商业帝国
- 致命弱点：功利、傲慢、心软

**金手指设计**：「商道三绝」
- 获得方式：穿越携带现代商业知识+历史先知
- 核心能力：信息战、金融术、供应链管理
- 使用代价：蝴蝶效应导致预知失效、身份风险
- 成长路径：汴梁小贩 → 跨州商号 → 天下商盟 → 经济命脉掌控者

**核心冲突**：
- 外部冲突：主角 vs 乱世的一切（战乱、军阀、混乱本身）
- 内部冲突：现代价值观 vs 乱世生存法则

### 6.7 创意方向C - 品类选择 [S01-BATCH-04]

**确定品类**：玄幻仙侠 + 规则怪谈 + 诡异流

**细分定位**：以"修仙世界遵守诡异规则才能存活"为核心设定

**选择理由**：
- 玄幻仙侠基本盘稳定（22.7%），规则怪谈增长迅猛（180%）
- "诡异仙侠"几乎是处女地，差异化空间极大
- 规则设计是AI优势领域，可保证逻辑自洽

### 6.8 创意方向C - 核心创意设计 [S01-BATCH-04]

**一句话卖点**：无灵根杂役误入诡仙界，唯凡人之躯可破禁忌，一路逆伐成仙。

**主角设定**：
- 姓名：林守一
- 身份：青云宗杂役弟子，无灵根、无背景、无资源的"三无"修士
- 核心欲望：活下去，活得像个人
- 致命弱点：过度谨慎、不信任他人、执念深重

**金手指设计**：「规则免疫」
- 获得方式：无灵根意味着无灵气，规则"找不到"他
- 核心能力：规则无视、规则感知、规则传递、规则破解
- 使用代价：无法修炼、时间限制、认知负荷、身份暴露风险
- 成长路径：窥规者 → 解规者 → 破规者 → 逆规者

**核心冲突**：
- 外部冲突：主角 vs 诡仙界的一切诡异规则
- 内部冲突：凡人的局限性 vs 仙道的诱惑（是否吸收诡异获得力量）

### 6.9 三方案评估决策 [S01-BATCH-04]

**评估结果**：

| 方案 | 商业潜力 | 可扩展性 | 差异化 | 执行难度 | 兴趣优势 | 总分 |
|-----|---------|---------|-------|---------|---------|-----|
| A（灵气复苏×传统文化） | 8分 | 7分 | 9分 | 6分 | 8分 | 7.70 |
| B（历史架空×五代商战） | 7分 | 8分 | 8分 | 5分 | 7分 | 7.10 |
| **C（仙侠×规则怪谈）** | 9分 | 9分 | 10分 | 4分 | 9分 | **8.45** |

**胜出方案**：方案C（玄幻仙侠 × 规则怪谈）

**决策理由**：
1. 差异化优势无可比拟："诡仙界"概念前所未见
2. 商业潜力最为突出：覆盖仙侠+规则怪谈双重读者群
3. 可扩展性解决长篇痛点：规则可无限创造
4. AI优势明显：规则设计是逻辑推演强项

### 6.10 胜出方案一句话卖点 [S01-BATCH-04]

> **无灵根杂役误入诡仙界，唯凡人之躯可破禁忌，一路逆伐成仙。**

**四要素拆解**：
- 主角身份：无灵根杂役
- 核心困境：误入诡仙界
- 独特外挂：唯凡人之躯可破禁忌
- 终极目标：逆伐成仙

**3秒测试结果**：8/10愿意点开 ✅

### 6.11 主角深度人设 [S01-BATCH-05]

**基础信息**：
- 姓名：林守一
- 年龄：19岁（故事开始时）
- 出身：青云宗外门杂役弟子，无灵根、无背景、无资源
- 外貌：身形偏瘦，眉骨较高，眼窝深陷，瞳色偏淡。右手虎口有旧疤。

**性格标签**：
- 核心优点：极致谨慎、冷静自持、坚韧不屈
- 致命缺陷：过度不信任、执念深重、自我封闭

**心理侧写**：
- 童年经历：七岁时村落被妖兽袭击，父母双亡，被青云宗收容做杂役
- 核心恐惧：无意义的死亡、被遗忘
- 终极欲望：证明凡人的价值
- 道德底线：不背叛恩人、不伤害弱者、不违背承诺

**成长弧光**：
- 起点：封闭、防备、低自尊
- 中期：尝试信任、被背叛、再度挣扎
- 终点：学会信任、为他人而战、开创凡道

### 6.12 核心冲突体系 [S01-BATCH-05]

**外部冲突三层结构**：
- Lv1：主角 vs 具体规则（识别→理解→破解）
- Lv2：主角 vs 诡异生物（利用规则漏洞击败）
- Lv3：主角 vs 诡神（摧毁诡仙界核心）

**内部冲突**：凡人的局限性 vs 仙道的诱惑（诡化 vs 保持人性）

**终极对手**：虚无诡神（归墟之主）
- 身份：诡仙界最高层级的诡神，所有规则的源头
- 动机：将整个修仙界都变成诡仙界
- 不可战胜性：力量差距、主场优势、信息不对称、不死特性
- 战胜方式：利用创世规则漏洞，摧毁规则之源

### 6.13 外挂体系规则 [S01-BATCH-05]

**金手指名称**：规则免疫

**能力层级**：
- Lv1 规则无视：被动免疫灵气锚定型规则
- Lv2 规则感知：主动探测规则存在
- Lv3 规则传递：将免疫临时传递给他人
- Lv4 规则破解：主动干扰和破坏规则

**成长境界**：窥规者 → 测规者 → 渡规者 → 解规者 → 逆规者

**使用代价**：凡人体质脆弱、精神消耗、传递削弱自身、暴露风险

**上限设定**：
- 创世规则不可破
- 诡神不可直接对抗
- 凡人寿命限制
- 物理/精神攻击有效

### 6.14 核心爽感循环 [S01-BATCH-05]

**五步循环模型**：
1. 陷入危机/被轻视 → 建立期待
2. 识别规则/展示能力 → 展现智慧
3. 打脸/逆转 → 情绪高潮
4. 获得资源/变强 → 奖励反馈
5. 引出新危机 → 开启下一轮

**循环频率**：
- 微循环：1-2章（小爽点）
- 中循环：5-8章（完整五步）
- 大循环：1卷（打败重要敌人）
- 终极循环：全书（打败最终BOSS）

### 6.15 目标读者画像 [S01-BATCH-05]

**核心读者**：
- 年龄：18-30岁（占比65%）
- 性别：男性为主（75%）
- 地域：新一线/二线城市为核心
- 职业：学生、互联网/IT、白领
- 付费：月均30-100元
- 阅读时长：日均60分钟以上

**一句话定位**：喜欢智斗推理、期待逆袭爽感、追求设定严谨的18-30岁男性网文读者
